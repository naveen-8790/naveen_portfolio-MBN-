Electricity Generation with Piezoelectric Discs

Project Overview

This project explores sustainable energy generation using piezoelectric materials that convert mechanical pressure into electrical energy. The system is designed for high-traffic areas to harness energy from footsteps and other pressure sources.

Key Components

  • Piezoelectric discs arranged in a matrix configuration
  • Energy harvesting circuit with voltage rectification
  • Energy storage system using supercapacitors
  • Power management unit for consistent output
  • LED indicators for system status monitoring

Technologies Used

Piezoelectric Materials Circuit Design Energy Harvesting Power Electronics Supercapacitors
